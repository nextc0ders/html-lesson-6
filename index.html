1. Attributes

HTML attributes are used to add additional information and behavior to elements.

rel

The rel (relationship) attribute specifies the relationship of the link to the current document. It is used in the <a> and <link> tags.

rel="noopener noreferrer"

Used in conjunction with target="_blank" to improve security and performance.

  • noopener — prevents new tab access to window.opener
  • noreferrer — does not transmit referral source information

aria-labelledby

The aria-labelledby attribute links an element to another element on the page, giving it an accessible name. This is used if the heading or text already exists.

aria-describedby

The aria-describedby attribute adds additional description of the element for screen readers. It is used for explanations and instructions.

aria-label

The aria-label attribute directly specifies the accessible name of an element. It is only used if the element has no visible text.

4. Meta tags

Meta tags are located in the <head> section and provide information about the page for browsers, search engines, and other services.

  • <meta charset="UTF-8"> — character encoding
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> — responsive layout
  • <meta name="description"> — page description
  • <meta name="keywords"> — keywords
  • <meta name="author"> — author
  • <meta name="robots" content="index, follow"> — Tells search engines whether the page should be indexed and whether links should be followed.
  • <meta name="theme-color" content="#ffffff"> — Sets the browser theme color on mobile devices.
